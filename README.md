#BRAIN BLITZ-MIND VAULT GAME
The user discovers digits by matching cards in a memory game.
The 8 digits are collected in sequential order and the user must enter the
correct 8-digit code from the revealed numbers to finish the game.

This project has 3 files
#index.html
  Defines the layout of the Game 
#script.js
  This has working logic for the game.
#styles.css
  It included style properties for the game.
