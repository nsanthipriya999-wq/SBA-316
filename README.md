#BRAIN BLITZ-MIND VAULT GAME

A fun memory-matching game built using HTML, CSS, and JavaScript.

The goal is simple: Match all the card pairs to reveal clues. Each clue gives you one digit of a secret 8-digit code. Once you've found all the clues, enter the correct code to unlock the vault and win the game.
TIP:
The 8 clues are listed in sequential order and the user must enter the
correct 8-digit code from the revealed clues to finish the game.

This project has 3 files 
#index.html
  Defines the layout of the Game 
#script.js
  This has working logic for the game.
#styles.css
  It included style properties for the game.
